# 贪吃蛇小游戏

一个纯前端实现的贪吃蛇小游戏，使用原生 `HTML`、`CSS` 和 `JavaScript` 编写，无需安装依赖。

## 玩法

- 使用方向键或 `WASD` 控制蛇移动
- 吃到绿色果实后加分，蛇身会变长
- 撞到边界或自己的身体时游戏结束
- 支持暂停、重开和移动端方向按钮

## 功能

- 响应式布局，适配桌面和手机
- Canvas 绘制游戏画面
- 本地最高分记录
- 暂停 / 继续 / 重新开始

## 如何运行

直接用本地静态服务器打开项目即可。

```bash
cd /Users/lm/git/snake-game
python3 -m http.server 4173
```

然后在浏览器访问：

```text
http://localhost:4173
```

## 操作说明

- `方向键` 或 `WASD`：移动
- `空格`：暂停 / 继续
- `R`：重新开始
- 手机端：点击页面下方的方向按钮

## 项目结构

- `index.html`：页面结构
- `styles.css`：界面样式
- `game.js`：游戏逻辑
- `design/`：设计参考和验收截图

## 备注

如果要直接部署到 GitHub Pages，这个项目可以作为静态站点使用。
